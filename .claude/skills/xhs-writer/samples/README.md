## 🎯 风格推荐逻辑

**核心原则：Felix偏好深度论述，而非极简列表。**

根据用户的话题类型，推荐相应的风格参考：

### 场景 → 风格映射表

| 用户话题关键词 | 推荐样本ID | 写作模式 | 核心特点 |
|--------------|-----------|---------|---------|
| 商业战略、投资、资本、品牌升级、竞争分析 | #1 a16z投资逻辑 | 深度论述 | 多维度拆解 + 案例密集 + 商业类比 |
| 职场选择、个人发展、大厂、创业、AI从业 | #2 大厂vs创业 | 深度论述 | 对比框架 + 真实痛点 + 连贯段落 |
| 技术趋势、行业观点、AI工具、技术选型 | #3 Agent vs Workflow | 深度论述 | 冷静拆解 + 技术类比 + 反潮流观点 |
| 品牌营销、出海策略、案例分析、用户心理 | #4 跨境品牌性格 | 快速共鸣 | 案例三连 + 落地指南 + 简洁有力 |
| 个人成长、哲学思考、兴趣变现、长文 | #5 多重兴趣 | 深度论述 | 长文深度 + 系统性框架 + 哲学思考 |

**两种写作模式的区别：**
- **深度论述模式（推荐）：** 6-8个维度拆解 + 连贯段落 + 技术/商业类比。适合B端决策者，展示思考深度。
- **快速共鸣模式：** 4步公式 + 场景案例 + 行动指南。适合快速输出，情绪共鸣优先。

## 📖 使用指南

**AI使用流程：**
1. 在 SKILL.md 的 Step 2.5（新增）中，分析用户话题并推荐2-3个风格选项
2. 用户选择风格后，读取对应的完整文章
3. **不要一次性读取所有文件**，每次只读取1个样本

**风格推荐话术示例：**
```
基于你的话题"XX"，我推荐以下写作风格：

1️⃣ 深度论述风格（参考：Agent vs Workflow）
   - 适合：战略级话题、需要展示架构思维
   - 特点：6-8维度拆解、连贯段落论述、技术类比、更像"Felix在思考"

2️⃣ 快速共鸣风格（参考：大厂vs创业）
   - 适合：职场话题、快速输出、情绪共鸣优先
   - 特点：对比框架、场景案例、行动指南

推荐：深度论述风格（更符合你的人设和目标客户）

你想用哪种风格？
```
